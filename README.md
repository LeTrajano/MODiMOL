# MODiMOL

**Modelagem para Inovação Molecular**

Plataforma computacional para validação, padronização, caracterização e análise de estruturas moleculares utilizando RDKit, FastAPI e Ketcher.

---

## Objetivo

O MODiMOL foi desenvolvido para fornecer uma infraestrutura computacional destinada ao processamento de moléculas químicas, permitindo:

- desenho molecular;
- obtenção automática de SMILES;
- normalização estrutural;
- geração de InChI e InChIKey;
- cálculo de propriedades moleculares;
- busca por moléculas similares;
- consulta ao nome IUPAC via PubChem;
- persistência em banco de dados.
